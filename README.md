# Landing Page UMKM Template

Template landing page sederhana untuk usaha, UMKM, dan personal branding menggunakan HTML dan Tailwind CSS.

Template ini digunakan dalam workshop **"Membuat Landing Page UMKM dengan HTML dan Tailwind CSS"** yang diselenggarakan oleh STARA.

---

## Fitur

* Responsive (mobile & desktop)
* Menggunakan Tailwind CSS (tanpa instalasi)
* Mudah dikustomisasi
* Cocok untuk:

  * Toko makanan
  * Laundry
  * Salon
  * Jasa desain
  * Kursus privat
  * Konsultan
  * UMKM lainnya

---
## Struktur Repository
.
├── index.html
├── README.md
└── assets/


Keterangan

* `index.html` → Halaman utama landing page.
* `README.md` → Dokumentasi penggunaan template.
* `assets/` → Folder untuk menyimpan gambar, logo, dan aset lainnya yang digunakan pada website.

Peserta dapat menambahkan gambar produk, logo usaha, atau foto bisnis ke dalam folder `assets/` kemudian menggunakannya pada halaman website.

## Cara Menggunakan Template

# 1. Gunakan Template Repository
- Klik tombol **Use this template** pada halaman repository GitHub.
- Pilih: Create a new repository
- Beri nama repository sesuai keinginan
- Klik **Create repository**. GitHub akan membuat salinan project ke akun GitHub Anda.

# 2. Salin URL Repository
- Buka repository yang baru dibuat.
- Klik tombol **Code** lalu salin URL repository.
- Contoh: https://github.com/username/toko-kue-saya.git

# 3. Clone Repository ke Komputer
- Buka Visual Studio Code.
- Tekan: Ctrl + Shift + P
- Cari: Git: Clone
- Tempel URL repository yang telah disalin.
- Pilih folder penyimpanan project.
- Setelah proses selesai, klik: Open

# 4. Edit Isi Landing Page
- Buka file: index.html
- Kemudian ubah isi sesuai kebutuhan
- Gambar dapat disimpan pada folder: assets/

# 5. Melihat Hasil Website
- Install extension: Live Server
- Kemudian klik kanan pada file: index.html
- Pilih: Open with Live Server
- Website akan terbuka di browser dan otomatis diperbarui setiap kali file disimpan.

atau 

- Klik ctrl + f5 atau buka menu "Run" -> dan klik "Run without debugging"
- website akan bisa ditampilkan di browser secara lokal


# 6. Upload Perubahan ke GitHub
- Buka menu Source Control pada Visual Studio Code.
- Kemudian tulis pesan commit
- Klik Commit
- Klik Sync Changes atau Push
- Perubahan akan tersimpan pada repository GitHub.

---

## Publish ke GitHub Pages

# 1. Buka Settings
- Masuk ke menu: Settings → Pages

# 2. Aktifkan GitHub Pages
- Pada bagian Source pilih: Deploy from a branch
- Kemudian pilih:
  Branch: main
  Folder: /root
- Klik Save.

# 3. Akses Website
- Tunggu beberapa saat hingga proses deployment selesai.
- Website akan tersedia pada alamat: https://username.github.io/nama-repository
- Contoh: https://username.github.io/toko-kue-saya
